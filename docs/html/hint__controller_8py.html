Detailed Description

This program elaborates all the received hints.

Author
Riccardo Zuppetti
Version
1.0
Date
15/11/2021

Subscribes to:
/hint

Publishes to:
/hypo

Services:
None

Client Services:
armor_interface_srv

Action Services:
None

Description:
The node subscribes to the /hint topic, in order to operate on them. If the hint is not already present in the ontology, it adds to this one. If it is the first time the hint has been received, it checks if the hypothesis that correspond to that hint ID is complete and not inconsistent. In case the hypothesis is correct it sends it to the robot.
